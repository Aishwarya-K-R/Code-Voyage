*****Sudoku Problem Solver*****

Overview of the project :
This project is aimed at developing a simple web application which solves the classical Sudoku game.

Rules of the Sudoku game :
1. Same number shouldn't appear more than once in the same row, column and grid.
2. The player is supposed to enter any three random numbers in any cells of the 9X9 box. 
3. According to the numbers entered by the player, the box is filled with the appropriate numbers.  
4. In case the conflicting numbers are entered by the player, an appropriate error message is displayed.

Steps for the implementation :
1. Download the source code of the project.
2. Run index.html file in IDE - Visual Studio Code to view the output.
