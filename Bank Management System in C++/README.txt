*****Bank Management System Project in C++*****

This project simulates an actual Bank operations in an Online mode to the customers.

Benefits :
People literally understood the value of Online mode during the pandemic.
This project aims to benefit the customers by saving their time and energy going to the bank, giving an access to the bank operations from the place they are currently at. 

Functionalities implemented :
1. Creation of Bank Account with an automatic assignment of unique Account number for each Account. 
2. Updation of Bank Account
3. Deposit of Amount
4. Withdrawal of Amount
5. Viewing the Balance Amount in the Account
6. Viewing the Account Holder's Details
7. Closing an Account

Steps for Implementation :
1. Download the Source Code of the project.
2. Run the code in any IDE - Visual Studio Code, etc.
